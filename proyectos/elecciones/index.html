Proyecto Elecciones Presidenciales

Análisis de datos electorales con herramientas avanzadas de visualización

Nota: Este proyecto no tiene fines políticos. Su objetivo es mostrar el funcionamiento de la herramienta.

Sobre el Proyecto

Este proyecto analiza los resultados de elecciones en distintas regiones, utilizando metodologías de ETL y visualización de datos. Se recopilaron datos de múltiples fuentes oficiales a través de CSV descargados de página oficial y se procesaron para obtener insights clave sobre tendencias y participación electoral. (2015 -2023 Contextos similares )

Dashboard Interactivo

Resultados Destacados

  • - Identificación de patrones de votación por provincia.
  • - Predicción de tendencias electorales basadas en datos históricos.
  • - La tasa de participación disminuyó con respecto a la elección de 2015.
